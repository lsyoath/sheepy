The R-fMRI Course V2.1

(Mostly recorded at the Resting-State fMRI Free Webinar Course, Jan 25th, 2016)

I. Resting-State fMRI: Algorithms, Applications to Brain Disorders and Data Processing             

   Or Dr. Yu-Feng Zang's talk: Brief Introduction of Resting-State fMRI                 

II. Resting-State fMRI: Principles of Data Processing and Introduction to DPARSF                 

III. Resting-State fMRI: Quality Control, Standardization, Statistical Analysis, Results Viewing, The R-fMRI Maps Project and Introduction to DPABI                

IV. Introduction to DPARSF Basic Edition                

 

 

------------------------------------------------------------------------------------------------------------------------

The R-fMRI Course was firstly released on July 15, 2009. During the past 6 years, the course was kept updating, and received more than 110,000 views.

I have released the R-fMRI Course V2.1 as above, however, if you want to look at the R-fMRI Course V1.0, please see below.

------------------------------------------------------------------------------------------------------------------------

I. Multimedia Course: Data Processing of Resting-State fMRI (Part 1, English version) (Corresponding PowerPoint pps file) Release 091227:
1. How to use REST DICOM Sorter to arrange your data.
2. How to use SPM to preprocess your data.
3. How to use REST to detrend and filter your data, and how to use REST to calculate Functional Connectivity (FC), Regional Homogeneity (ReHo), Amplitude of Low-Frequency Fluctuation (ALFF), Fractional ALFF (fALFF).
4. How to use DPARSF to perform the one-stop analysis (from DCM2nii, preprocessing to FC, ReHo, ALFF, fALFF calculation).
5. How to use REST and DPARSF to regress out the nuisance covariates and extract ROI or AAL time courses.

I strongly recommend the new users download this multimedia course, this will keep you out of many unnecessary errors.

                                                               DOWNLOAD: Multimedia Course: Part 1, English version    PowerPoint pps


II. Multimedia Course: Data Processing of Resting-State fMRI (Part 2, English version) (Corresponding PowerPoint pps file) Release 091227:
1. How to use SPM to perform Statistical Analysis.
2. How to use REST Slice Viewer to view your results, set cluster size, make ROI mask files and create pictures suitable for publication.
3. How to use REST AlphaSim to perform multiple comparison correction.
4. How to use REST Image Calculator.
5. Other functions of REST: converting .nii files to .imd/.hdr files and REST's reading and writing functions.

                                                               DOWNLOAD: Multimedia Course: Part 2, English version    PowerPoint pps


III. Multimedia Course: Data Processing of Resting-State fMRI (Part 3 DPARSF V2.2 Advanced Edition, English version) (Corresponding PowerPoint file) Release 120923:
1. The different processing order in DPARSF V2.2 Advanced Edition.
2. The details of new functions in DPARSF V2.2 Advanced Edition.
3. The Template Parameters in DPARSF V2.2 Advanced Edition.

[Old Release (101109) of Part 3, English version(Corresponding PowerPoint pps file)]

IV.  Resting-State fMRI: Algorithms, Applications to Brain Disorders and Data Processing (English version) (Corresponding PowerPoint file) Release 130730:
This is a talk invited by Dr. Mariana Lazar at the Department of Radiology, New York University. I am talking about:

静息态功能磁共振数据处理教程自2009年7月15日发布以来，在过去的6年间，课程持续更新，并被访问11万余次。

静息态功能磁共振数据处理视频教程适用于REST，SPM和DPARSF的新使用者，也有助于理解一些静息态fMRI的基本概念。
一、静息态功能磁共振数据处理视频教程 (第一部分，中文版) (对应的PowerPoint pps 文件) Release 100820：
1. 使用REST DICOM Sorter整理原始数据。
2. 使用SPM对数据进行预处理。
3. 使用REST对数据进行去线性趋势、滤波等操作，以及使用REST进行Functional Connectivity (FC)，Regional Homogeneity (ReHo)，Amplitude of Low-Frequency Fluctuation (ALFF)和Fractional ALFF (fALFF)计算。
4. 使用DPARSF进行批处理分析(从DICOM数据到预处理，再到FC，ReHo，ALFF，fALFF结果)。
5. 使用REST和DPARSF去除协变量，提取感兴趣区(ROI)或AAL脑区的时间序列。

强烈建议新使用者下载本视频教程，可以帮助避免很多数据处理过程中的不必要错误。

                                                                                                      下载：视频教程：第一部分，中文版   PowerPoint pps

二、静息态功能磁共振数据处理视频教程 (第二部分，中文版) (对应的PowerPoint pps 文件) Release 100820：
1. 使用SPM和REST进行统计分析。
2. 使用REST Slice Viewer查看结果，设定cluster size，制作ROI mask，保存图片用于发表。
3. 多重比较校正：Monte Carlo模拟与Bonferroni、FDR、FWE校正的区别，以及REST AlphaSim的使用。
4. REST Image Calculator的使用。
5. 其他功能：.nii数据的转换，REST读写接口的使用。

                                                                                                      下载：视频教程：第二部分，中文版   PowerPoint pps

三、静息态功能磁共振数据处理视频教程 (第三部分，中文版) (对应的PowerPoint ppt 文件) Release 130425：
1. DPARSF V2.2 Advanced Edition中的不同数据处理顺序。
2. DPARSF V2.2 Advanced Edition新功能介绍。
3. DPARSF V2.2 Advanced Edition的各模板参数。

旧版本(Release 101219)：(第三部分，中文版) (对应的PowerPoint pps 文件)

                                                                                                      下载：视频教程：第三部分，中文版   PowerPoint ppt


四、静息态功能磁共振基本概念、计算方法及其在脑疾病中的应用视频 (中文版) (对 应的PowerPoint pps 文件) Release 100506：
1. 静息态功能磁共振数据处理教程的总体介绍。
2. 静息态功能磁共振的基本概念。
3. 静息态功能磁共振的常用计算方法。
4. 静息态功能磁共振在脑疾病中的应用。
NOTE: 该视频除总体介绍部分外，其余基于臧玉峰老师的报告PPT(少许增改)讲解，概念以臧老师的报告为准，推荐下载臧老师报告视频。本部分于重庆医科大学附属第一医院神经内科录制，特此感谢王学峰教授的邀请以及方维东、胡以达的大力帮助!

                                                                                                      下 载：总概视频 中文版   PowerPoint pps

2. 使用REST Slice Viewer截图。
3. 使用Photoshop对图片进行加工。

六、SCI论文写作——我的经验教训视频 (中文版) (对 应的PowerPoint pps 文件) Release 100824：
本视频是受广东省中医院总院刘波主任之邀，介绍了本人在SCI论文写作中的一些经验和教训。这些经验和教训不一定正确，也许对大家有些参考价值。视频中以处女作PLoS ONE文章和第二篇Cerebral Cortex文章为例，介绍了在文章选题、实验设计、数据处理、论文写作(Title / Abstract / Introduction / Methods / Results / Discussion / Conclusion / References)以及按审稿人意见修改等方面的经验。希望对大家有所帮助!

                                                                                                      下 载：SCI论文写作——我的经验教训视频视频 中文版   PowerPoint pps

七、Caret的简单使用视频 (中文版) Release 110410：
1. 本视频是关于如何将fMRI的统计结果映射到皮层表面进行显示的简要介绍。
2. 请到Caret网站下载Caret主程序（注意最新版与中文windows不太兼容，需要在控制面板->区域和语言选项->高级->非Unicode程序语言 选择英文，故亦可下载早前版本）和CARET_TUTORIAL_SEPT06.zip。
3. 下载视频中提到的两个设置文件：Yan_fMRIMapping.scene 和 Yan_fMRIMapping.spec （或者Caret 5.64兼容版本：Yan_fMRIMapping_Caret564.scene和Yan_fMRIMapping_Caret564.spec）并放到CARET_TUTORIAL_SEPT06目录下。
4. 更详细的说明，请参阅Caret网站上的相关文档。
